🔰 Objetivos del Encuentro
  • Introducir el uso de JavaScript para validar formularios.
  • Comprender la manipulación del DOM (Document Object Model).
  • Aplicar las primeras validaciones de datos del lado del cliente.
📖 Materiales
  • 📄 Guía práctica: Validación con JavaScript
  • 🎥 Video explicativo: Manipulación del DOM
  • 📄 Plantilla HTML con formulario base
🧩 Actividades
  • Agregar validaciones al formulario HTML usando JavaScript.
  • Manipular el DOM para mostrar mensajes de error o éxito.
  • Probar distintos casos de entrada de datos.
✅ Entregables
  • Formulario funcional validado con JS (mínimo 3 campos validados).
  • Entrega del archivo HTML con código JS integrado.
